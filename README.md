## Features

- **Baidu Pan** — Browse files or paste share links to download
- **Bilibili** — Download videos by pasting video URLs
- **Quark Pan** — Download from Quark share links
- **Clipboard auto-detect** — Paste any supported link and the app opens the right dialog
- **Priority download** — Right-click a queued item → "Download First"
- **Per-service speed limits** — Set independent speed limits for each service

## How to use

1. Run the app
2. Go to **Settings** and enter your credentials:
   - **Baidu Pan** — `BDUSS` (F12 → Application → Cookies → `BDUSS` on pan.baidu.com)
   - **Bilibili** — `SESSDATA` (F12 → Application → Cookies → `SESSDATA` on bilibili.com)
   - **Quark Pan** — `Cookie` (F12 → Network → any request → Headers → Cookie on pan.quark.cn)
3. Browse files or paste a share link → select files → download
